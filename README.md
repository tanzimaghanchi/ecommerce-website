# FAISHORA Project Structure

This is the active FAISHORA ecommerce project.

## Main folders

- `frontend` - React customer storefront and admin UI
- `backend` - Express + MySQL backend API

## Which project to run

Frontend:
- Path: `C:\final project\E-Commerce\frontend`
- API base URL: `http://localhost:5001`

Backend:
- Path: `C:\final project\E-Commerce\backend`
- Port: `5001`

## Notes

- The active database name is `ecommerce`.
- Keep all FAISHORA work inside this `C:\final project\E-Commerce` folder pair.
